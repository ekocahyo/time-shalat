# time-shalat
Waktu shalat bisa di kombinasi dengan conky

usage: python run.py [-h] [-s] [-d] [-a] [-m] [-i]

optional arguments:
-h, --help     show this help message and exit
-s, --subuh    Waktu Subuh
-d, --duhur    Waktu Duhur
-a, --asr      Waktu Asr
-m, --maghrib  Waktu Maghrib
-i, --isha     Waktu Isha
